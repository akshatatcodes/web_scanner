The Real Challenges
Building this was NOT easy — here's what we fought through
🔥
BullMQ Job Stalling
Long scans blocked the event loop, causing lock renewal failures. Fixed with async yielding and heartbeats.
🛡️
WAF Evasion
Cloudflare & Akamai blocked every scan. Built a strategy engine: URL encoding, double-encoding, fragmentation.
🔒
SSL Certificate Hell
"unable to get local issuer certificate" on Render cloud. Solved with a custom HTTPS agent bypass.
🐳
Puppeteer in Docker
Browser crashed inside containers. Isolated full crawl to an on-demand "Deep Scan" mode.
🤖
False Positives in CVE Matching
Too much noise from keyword matching. Integrated Gemini 2.5 AI to score confidence per finding.
💥
AggregateError Crashes
Raw Node.js stack traces dumped to the UI. Fixed with structured error extraction and clean messages.
